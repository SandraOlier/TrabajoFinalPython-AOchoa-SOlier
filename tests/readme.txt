📁 Carpeta de pruebas - Tests

Esta carpeta contiene scripts de prueba que se usaron durante el desarrollo de la aplicación.

✔️ test_datos.py  
Verifica que las funciones de acceso a la base de datos (cargar, insertar, eliminar) funcionen correctamente.

✔️ test_dbpath.py  
Prueba que la ruta hacia la base de datos mascotas.db esté bien construida, para evitar errores al mover archivos o cambiar carpetas.

⚠️ IMPORTANTE:  
Estos archivos NO forman parte de la aplicación final que verá el usuario. Son solo herramientas internas para desarrolladoras.
